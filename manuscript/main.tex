% ---------------------------------------------------------
% Abstract
% ---------------------------------------------------------

\abstract{Using simulation methods, we show that even a preregistered analysis of a single hypothesized independent variable can produce false positive findings at a much higher rate than the expected 5\%. By including interactions between the hypothesized variable and covariates, omitting main effects when estimating interactions, using dummy coded variables, and surprisingly by using large sample sizes, simple regression analyses sometimes produce type I errors exciting 35\% of the time. Using combinatorics, we show that including three covariates generate 94 possible models, using 6 over 2.3 million models. Systematically exploring a model set with just 3 covariates nearly guarantees a false positive finding. While labeling unregistered analyses as exploratory is an important practice, interpreting their p-values may still be meaningless.             
}

\keywords{type I error, false positive statistics, replicability, simulation}
