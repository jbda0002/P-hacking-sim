\title{How (even imperfect preregistered) research can produce false positive findings}

% ---------------------------------------------------------
% Abstract
% ---------------------------------------------------------

\abstract{Using simulation methods, we show that even a preregistered analysis with a single variable of interest which has no true effect can produce false positive findings at a much higher rate than the expected 5\%. To better understand the nature of false positive findings, we focus both on the probability that one model has a false positive result (the false positive probability) but also the number of models with a false positive result (the false positive rate). By omitting main effects when including interactions between a binary variable of interest and binary covariates, simple regression analyses can produce a false positive rate exciting 42\%. Using combinatorics, we show that collecting three covariates is enough to generate 94 possible models, whereas collecting six can produce over 2.3 million models. Having such large model sets nearly guarantees a false positive finding. Nevertheless, exploratory analyses should remain an important practice, interpreting their p-values might be meaningless considering the high probability of observing one model with a false-positive finding.             
}
